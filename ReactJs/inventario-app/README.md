# Inventario App

Este proyecto es una aplicación web, que permite llevar un control completo de Inventario, Entradas y Salidas, incluyendo creación, edición, eliminación y exportación de datos filtrados en PDF. Utiliza React con Vite como paquete, tailwindcss para estilos, cuenta con autenticación mediante Google y almacenamiento en Google Sheets.

## Features

- Autenticación con Google (Google Auth).
- Creación de archivo en Google Sheets.
- Agregar, modificar, eliminar y visualizar datos en una tabla.
- Exportación a PDF basada en filtros aplicados.

## Technologies Used

- React: Una biblioteca de JavaScript para construir interfaces de usuario.
- Vite: Una herramienta de construcción de aplicaciones web rápida y moderna para proyectos basados en JavaScript y TypeScript.
- Tailwind CSS: Un marco de diseño CSS de utilidad de bajo nivel para construir rápidamente diseños personalizados.

## Dependencies

- `react`: Biblioteca de JavaScript para crear interfaces de usuario.
- `react-dom`: Punto de entrada al DOM y a los renderizadores del servidor para React.
- `vite`: Entorno de desarrollo y bundler moderno.
- `zustand`: Manejador de estado global simple y escalable.
- `gapi-script`: Cargador del cliente de Google API (Auth, Sheets).
- `jspdf`: Generación de archivos PDF.
- `jspdf-autotable`: Generación de tablas dentro del PDF.
