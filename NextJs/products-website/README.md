# Products Website

Este proyecto es una página web que muestra la información de una empresa, su catálogo de productos y contenido informativo mediante artículos, de forma visual y estructurada. Desarrollada con Next.js y Tailwind CSS para los estilos.

## Features

- Página de destino moderna y responsiva.
- Barra de navegación superior.
- Sección de información de la empresa.
- Sección de productos.
- Sección de artículos o contenido informativo.
- Diseño adaptable a dispositivos móviles.
- Componentes reutilizables.
- Optimización básica para SEO.

## Technologies Used

- Next.js: Framework de React para crear aplicaciones web modernas y optimizadas.
- React: Una biblioteca de JavaScript para construir interfaces de usuario.
- TypeScript: Tipado estático para JavaScript.
- Pages Router: Sistema de enrutamiento de Next.js basado en archivos, con soporte para SSR y generación estática.
- Tailwind CSS: Framework de CSS orientado a utilidades para crear diseños modernos y responsivos de forma ágil y escalable.

## Dependencies

- `next`: Paquete principal que proporciona el framework Next.js.
- `react`: Biblioteca de JavaScript para crear interfaces de usuario.
- `react-dom`: Punto de entrada al DOM y a los renderizadores del servidor para React.
- `lucide-react`: Conjunto de iconos SVG optimizados para React.
