# Odoo Website

Este proyecto es una página web pensada únicamente para prueba y exploración, que permite a los usuarios explorar las funcionalidades disponibles de Odoo 19 en su versión gratuita, sin necesidad de instalarlo localmente. Desarrollada con Next.js, TypeScript y Tailwind CSS para los estilos.

## Features

- Interfaz moderna, simple y accesible para demostraciones.
- Selección de usuarios de demostración.
- Visualización de Odoo 19 Community.

## Technologies Used

- Next.js: Framework de React para crear aplicaciones web modernas y optimizadas.
- React: Una biblioteca de JavaScript para construir interfaces de usuario.
- TypeScript: Tipado estático para JavaScript.
- App Router: Sistema de rutas moderno de Next.js.
- Tailwind CSS: Framework de CSS orientado a utilidades para crear diseños modernos y responsivos de forma ágil y escalable.

## Dependencies

- `next`: Paquete principal que proporciona el framework Next.js.
- `react`: Biblioteca de JavaScript para crear interfaces de usuario.
- `react-dom`: Punto de entrada al DOM y a los renderizadores del servidor para React.
- `react-icons`: Librería de íconos SVG para React.
